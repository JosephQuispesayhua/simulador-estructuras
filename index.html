Simulador de Estructuras de Datos

Estructura de Datos y Algoritmos · 2026-A

Sobre el proyecto

Este simulador permite visualizar de forma gráfica el funcionamiento de cuatro estructuras de datos fundamentales: Pila, Cola, Lista enlazada y Árbol binario. Cada módulo muestra las operaciones de inserción y eliminación paso a paso (con algunas operaciones adicionales), junto con una explicación en texto de lo que ocurre en cada acción.

Docente: Roni Guillermo Apaza Aceituno

Integrantes: Fabrizio Alonso Quispe Rupaylla, Joseph Brayan Quispesayhua Hancco, Limberg Sarmiento Tico, Abel Alexander Huayta Choquepata

Estructuras disponibles

Pila (Stack)

Push y Pop — estructura LIFO.

Cola (Queue)

Enqueue y Dequeue — estructura FIFO.

Lista enlazada

Insertar y eliminar nodos enlazados.

Árbol binario

Insertar y eliminar nodos del árbol.
