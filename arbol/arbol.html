¿Qué es un Árbol Binario?

Un árbol binario de búsqueda (BST) organiza los datos de forma jerárquica. Cada nodo puede tener como máximo dos hijos:

  • Hijo izquierdo → valores menores.
  • Hijo derecho → valores mayores.

Cada operación mostrará gráficamente cómo cambia el árbol.
